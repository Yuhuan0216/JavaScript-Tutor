# JavaScript-Tutor

### 首先 工具跟設備是必不可少的
#### VSCode 推薦插件:
- Chinese Traditonal        => 繁體中文介面
- JavaScript and TypeScript => JS, TS 語法工具
- Prettier                  => 排版工具
- ESLint                    => JS 語法工具
- codeium                   => 免費的 AI 程式碼自動工具
- Git Blame                 => 追蹤代碼提交紀錄, 抓戰犯用的
- Git Graph                 => 強化 VSCode Git GUI介面
- Swimm                     => AI 文檔生成工具
